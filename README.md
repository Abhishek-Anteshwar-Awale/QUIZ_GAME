# Quiz Game Application

## 📌 Project Description

The Quiz Game is an interactive application that allows users to test their knowledge by answering multiple-choice questions. The system displays questions one by one, allows the user to select answers, and calculates the final score based on correct responses.

This project demonstrates basic programming concepts such as conditional logic, user interaction, data handling, and application flow.

---

## 🚀 Features

* Multiple-choice quiz questions
* Real-time score calculation
* User-friendly interface
* Instant feedback after answering questions
* Simple and interactive gameplay

---

## 🛠️ Technologies Used

* Java
* Android Studio
* Gradle
* XML (for UI design)

## ⚙️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/your-username/QUIZ_GAME.git
```

2. Open the project in **Android Studio**

3. Let Gradle sync the project dependencies

4. Run the application on:

* Android Emulator
  or
* Physical Android Device

## 🎯 Purpose of the Project

The purpose of this project is to demonstrate the implementation of a quiz-based application using Android development tools. It helps in understanding application structure, user interaction, and logic implementation in mobile applications.
